# **Постановка задачи**

В нашем сообществе Joystream было замечено, что многие технически подкованные люди присоединяются к нам с компьютерами на базе операционной системы Windows, из-за отсутствия у них компьютеров с MacOS или Unix системой они не могут внести свой вклад в наше сообщество.

# Обзор

Подготовка компьютера с Windows к настройке среды разработки Joystream. Пользователи нашего сообщества с компьютерами на базе операционной системы Windows смогут начать работать.

## Процессы установки

1. **Установите WSL на Windows компьютер**
2. Установите операционную систему Ubuntu на Windows компьютер
3. Установите VS Code
4. Настройка WSL в VS Code
5. Клонируйте репозитории Joystream pioneer с GitHub
6. Запустите Joystream Pioneer на локальном хосте.

## 1. Установите WSL на Windows компьютер

Пожалуйста, следуйте этой документации, чтобы установить WSL на ваш компьютер.

[[https://docs.microsoft.com/en-us/windows/wsl/install](https://docs.microsoft.com/en-us/windows/wsl/install)] ([https://docs.microsoft.com/en-us/windows/wsl/install](https://docs.microsoft.com/en-us/windows/wsl/install))

# 2. Установите Ubuntu

Перейдите в Microsoft Store и найдите Ubuntu.

![Untitled](../../Past%20Bounties/Setting%20up%20Joystream%20Development%20Environment%20in%20Wi/Untitled%201.png)

После установки введите **sudo apt update && sudo apt upgrade**, чтобы завершить настройку.

![Untitled](../../Past%20Bounties/Setting%20up%20Joystream%20Development%20Environment%20in%20Wi/Untitled%202.png)

# 3. Установите VS Code

По ссылке ниже есть полное руководство по установке VS Code, пожалуйста, ознакомьтесь с ним и следуйте инструкциям указанным  там.

[https://code.visualstudio.com/docs/setup/windows](https://code.visualstudio.com/docs/setup/windows)

# 4. Настройка WSL в VS Code

Откройте VS Code, перейдите к поиску расширений и найдите Remote WSL.

![Untitled](../../Past%20Bounties/Setting%20up%20Joystream%20Development%20Environment%20in%20Wi/Untitled%203.png)

Установите его.

Теперь закройте VS Code и снова откройте.

Откройте окно New WSL из VS Code и откройте новое окно терминала.

- [*https://github.com/Joystream/pioneer#quickstart](https://github.com/Joystream/pioneer#quickstart) **внимательно ознакомьтесь с этим материалом и попытайтесь понять структуру**.

# 5. Клонируйте репозиторий Joystream pioneer с GitHub

Из терминала VS Code WSL клонируйте репозиторий Joystream Pioneer

[https://github.com/Joystream/pioneer.git](https://github.com/Joystream/pioneer.git)

![Untitled](../../Past%20Bounties/Setting%20up%20Joystream%20Development%20Environment%20in%20Wi/Untitled%204.png)

**Это** **очень важный шаг** после клонирования. Смените директорию на Pioneer.

Введите yarn и enter, после успешной установки вы увидите следующее.

![Untitled](../../Past%20Bounties/Setting%20up%20Joystream%20Development%20Environment%20in%20Wi/Untitled%205.png)

Теперь выполните сборку. Для этого закройте код VS, перейдите в Windows PowerShell, перейдите в каталог Pioneer и выполните yarn сборку.

## 6. Запустите Joystream Pioneer на локальном хосте.

############ Создайте файл .env и вставьте следующий код #############

REACT_APP_TESTNET_NODE_SOCKET=wss://rpc.joystream.org:9944 REACT_APP_TESTNET_QUERY_NODE=https://query.joystream.org/graphql REACT_APP_TESTNET_QUERY_NODE_SOCKET=wss://query.joystream.org/graphql REACT_APP_TESTNET_MEMBERSHIP_FAUCET_18.2=http://3.3.8.18.2=https://query.joystream.org/graphql .io/member-faucet/register

Добавьте файл .env в папку Pioneer/packages/ui.

Теперь выполните запуск yarn

[http://localhost:8080/#/settings](http://localhost:8080/#/settings) и выберите «Joystream-Testnet» в раскрывающемся списке!

![Untitled](../../Past%20Bounties/Setting%20up%20Joystream%20Development%20Environment%20in%20Wi/Untitled%206.png)

Вот и все, теперь вы можете начать работу со своего компьютера на Windows!
