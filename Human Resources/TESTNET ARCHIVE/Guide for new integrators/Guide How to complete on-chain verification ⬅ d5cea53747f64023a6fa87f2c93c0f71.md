### You need to go to the general channel and enter the command - /claim

![Untitled](Guide%20How%20to%20complete%20on-chain%20verification%20%E2%AC%85/Untitled.png)

### Now you have to select this command. You have the opportunity to enter your data.

/claim username:adovrntest wallet:5EbogUBLjHnfG4MoRiC2uT9e7jwS82bn3YMecnzkngTuT6TD

**username** - your membership here - [https://dao.joystream.org/#/profile/memberships](https://dao.joystream.org/#/profile/memberships). Copy it and paste it in without any spaces. This is important.
**wallet** - here you need to specify the address of your wallet. It is important to specify the ROOT ACCOUNT address.

![Untitled](Guide%20How%20to%20complete%20on-chain%20verification%20%E2%AC%85/Untitled%201.png)

### Send a command and get a message

Next, you need to click on [**"Polkadot App"](https://polkadot.js.org/apps/?rpc=wss://rpc.joystream.org:9944/#/signing)** in the message.

![Untitled](Guide%20How%20to%20complete%20on-chain%20verification%20%E2%AC%85/Untitled%202.png)

### Getting a special code

Once you are on the page you need to select the ROOT ACCOUNT from which you will sign the transaction.

Then you need to sign the following data in the "sign the following data" section and enter your code that you received in the message.

Next, sign message and copy the code you received.

![Untitled](Guide%20How%20to%20complete%20on-chain%20verification%20%E2%AC%85/Untitled%203.png)

### Then you need to go to the general channel again and enter the command - /solve

In the line that appears in the chat, you need to paste the copied code and send a message.

![Untitled](Guide%20How%20to%20complete%20on-chain%20verification%20%E2%AC%85/Untitled%204.png)

### Congratulations

After a while you will have the "on-chain identity verified" role and the group role
