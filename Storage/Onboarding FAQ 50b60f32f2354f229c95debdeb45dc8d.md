# Onboarding FAQ

### **How to get on board?**

Apply here  [https://dao.joystream.org/#/working-groups/openings](https://dao.joystream.org/#/working-groups/storage) once the Storage vacancy is available. If there is no vacancy, please stay tuned and follow announcements in the #storage Discord channel [https://discord.gg/2vEnPcxF](https://discord.gg/2vEnPcxF) 

### What are server requirements?

```
You have access to, or can quickly deploy, a "good" dedicated server with:
   - At Least a 25TB of SSD storage
   - 8core+
   - 32GB+ RAM
   - Good network connectivity (not less than 100Gbps for international traffic)

For reference see [configs](https://github.com/Joystream/community-repo/tree/master/working-groups/storage-group/NodeSteup) 

```
