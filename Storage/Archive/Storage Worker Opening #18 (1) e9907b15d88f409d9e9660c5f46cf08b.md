# Storage Worker Opening #18 (1)

### Storage **Worker Opening #18**

**Link** 

[https://dao.joystream.org/#/working-groups/openings/storageWorkingGroup-18](https://dao.joystream.org/#/working-groups/openings/storageWorkingGroup-18) 

 **Description**

We are looking for IT savvy candidate who is familiar with Linux and have experience of maintaining blockchain nodes. Storage provider run and maintain storage nodes that store very large quantities of static data, synchronize with other storage nodes, share data with distributors, and accepts inbound uploads from end users. 

**Requirements**

- Experience with Linux
- Experience of maintaining blockchain nodes
- Degree in CS is a plus
- Average workload is about 2-5 hours per week
- Every applicant should make a test task [https://www.notion.so/joystream/Bounty-Storage-WG-Entry-Level-74b13f81a32d4efb811f8259d6fbeee0](Bounty%20%5BStorage%20WG%5D%20%5BEntry%20Level%5D%2074b13f81a32d4efb811f8259d6fbeee0.md)

**Rewards:**

Reward Per Block 4 tJOY (will be 1 tJOY initially before node is fully set up and included to a distributor t)

**Server Specs for Reference** 

Storage Node has the following (minimal) specs:

- 4(v)CPU, 8GB RAM, 1500GB HDD and SSD. Or upper. Ubuntu 20.04
- Any hosting providers are acceptable unless the Storage node concentration don’t exceed 30%
- Usually it cost about 50-80$ to rent a server with the required specs
