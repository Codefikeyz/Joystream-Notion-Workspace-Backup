# Discord Bot Command

1. **Link your discord account to your Joystream account**

Type on discord this `/claim` and input your username and root wallet address

    
    ![Untitled](Discord%20Bot%20Command/Untitled.png)
    
    The result is a string like this
    `CIwuTOcOcf7w0zS3n9fsW`
    

 2.  **Sign with Polkadot App to prove your identity** 

Go to this URL

[https://polkadot.js.org/apps/?rpc=wss://rpc.joystream.org:9944#/signing](https://polkadot.js.org/apps/?rpc=wss://rpc.joystream.org:9944#/signing)

![Untitled](Discord%20Bot%20Command/Untitled%201.png)

Input the above string from discord and click “Sign message”
You will get a signature like this.
`0x362751d8fb030c39a88d95b90d432f24e75283ac1351d586b686b89cfe4b12371ef7cca3af4387824235a42b9df04c7c2dd6965d38508104cd1d72f7ddbbfa8e`

1. **Finish  link with bot command connection**
     
In discord type `/solve` and input the above signature.

![Untitled](Discord%20Bot%20Command/Untitled%202.png)

You will see your Joystream roles in discord profile like this.

![Untitled](Discord%20Bot%20Command/Untitled%203.png)

Finished!
