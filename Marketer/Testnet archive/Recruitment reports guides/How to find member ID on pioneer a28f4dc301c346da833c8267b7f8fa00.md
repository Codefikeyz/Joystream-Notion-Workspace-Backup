# How to find member ID on pioneer?

First find homepage here: [https://dao.joystream.org/](https://dao.joystream.org/) 

Now, you have interface like this, please click on “**My profile**”

![1663986934102.png](How%20to%20find%20member%20ID%20on%20pioneer/1663986934102.png)

Now, you have interface like this, please click on “**My memberships**”

![1663987026042.png](How%20to%20find%20member%20ID%20on%20pioneer/1663987026042.png)

When you click there, you will get your member ID here:

![1663987130788.png](How%20to%20find%20member%20ID%20on%20pioneer/1663987130788.png)
