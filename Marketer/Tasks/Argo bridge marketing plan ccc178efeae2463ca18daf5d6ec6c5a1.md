# Event Overview

Joystream has built a bridge from L1 to Base chain, allowing users to bridge their JOY tokens from the Joystream network to Base chain. The bridge is operated by the DAO through selected trusted multisig signers on both Base chain and Joystream chain.

# Marketing Plan

### 1. Multichannel Announcement

- **Platforms**: Joystream Blog, Twitter, Telegram, Discord, Emails, DeBank, Link3, Warpcaster, Reddit
- **Action Items**:
    - draft blog post explaining the bridge, its benefits, and how to use it
    - create announcements for each social media platform
    - release posts across all platforms for maximum visibility when AMM on Uniswap launched
    

### 2. Bounties for ambassadors

- **Objective**: Generate user-created educational content to increase organic reach
- **Bounty Tasks**:
    - YouTube video tutorials on using the bridge
    - Twitter threads explaining the benefits
    - Medium/Substack articles on the impact of the bridge on Joystream ecosystem
- **Action Items**:
    - Set up a bounty program within our ambassador program with clear guidelines and rewards
    - Manage submission and review process
    - Send rewards
    

### 3. Targeted Advertising (start 1 week after launch)

- **Platform**: Twitter Ads
- **Target Audience**: BASE community
- **Action Items**:
    - set up Twitter Ads campaigns with appropriate targeting
    - allocate budget and set daily spend limits (I suggest start with 50$ daily, monitor efficiency for 1 week, then increase/decrease if needed)
    

### 4. Web 3 Outreach

- **Objective**: Secure coverage in crypto and tech media outlets and catalogues
- **Action Items**:
    - Compile a list of relevant web3 media and journalists
    - Draft a press release highlighting the bridge launch and its implications
    - Execute outreach to key journalists and editors
    - list wrapped JOY on web3 catalogues, fulfil the project infoes
    - list wrapped JOY on trustwallet repo
        - [https://developer.trustwallet.com/developer/listing-new-assets/requirements](https://developer.trustwallet.com/developer/listing-new-assets/requirements)
    - list JOY on Aerodrome DEX AMM /  wubmit WL & engage with AERO team

### 5. Community Engagement
