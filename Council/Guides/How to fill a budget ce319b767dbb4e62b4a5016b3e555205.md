# How to fill a budget

**At the beginning of term on Monday after the Council Period Parameters updated Council needs fill the budget:**

![Untitled](How%20to%20fill%20a%20budget/Untitled.png)

- **Term Budget**
    - 100 M tJOY - this is the amount available
    
    ![Untitled](How%20to%20fill%20a%20budget/Untitled%201.png)
    
    - ($2150)  USD_SUBSIDY (2400) - 250 (SP & DP) = 2150
    
    ![Untitled](How%20to%20fill%20a%20budget/Untitled%202.png)
    
    - 2.5 M JOY ($150 k) - this is JOY allocation
    
    3.5 M = $210k (JOY_BUDGET)
    
    x=$180k (JOY_BUDGET)
    
    x=3.5*180k\210 k= 630k\210 k=3 M
    
    - $1500 / 1M tJOY
    
    1M tJOY = $150 k/ 100 M (tJOY budget) = $1500
    
    - Subsidy: SP ($100) | DP ($150)
    
    ![Untitled](How%20to%20fill%20a%20budget/Untitled%203.png)
    

- **Fill the tabular part**
    
    ![Untitled](How%20to%20check%20the%20amount%20in%20the%20WG%20budgets/Untitled.png)
    
    To fill the Start Budget (MtJOY) use the [guide](How%20to%20check%20the%20amount%20in%20the%20WG%20budgets%20863e2c26abb843cf80cba0f000783e02.md).
    
    - To calculate Lead Reward you need to go to this **[link](https://dao.joystream.org/#/working-groups)** and find **Reward** for every Lead WG
    
    ![Untitled](How%20to%20fill%20a%20budget/Untitled%204.png)
    
    - E.g. 202,440 *7 = ****1.4
    
    ![Untitled](How%20to%20fill%20a%20budget/Untitled%205.png)
    
    - The weight of the Working group is listed [**here](https://joystream.gitbook.io/testnet-workspace/testnet/council-period-scoring#weights)** in the section “Weight”. Just copy and put it in the spreadsheet
    
    ![Untitled](How%20to%20fill%20a%20budget/Untitled%206.png)
    

- **Validators**
    - Calculate Validators Rewards using the [guide](How%20to%20find%20the%20start%20and%20end%20block%20of%20the%20council%20cb6e33effa5a4b75b0ba8624cbd5e179.md). But using only **the block start** and not using the era reward average but the **era reward at the start of the term:**
    
     E.g. [10.34 M](How%20to%20find%20the%20start%20and%20end%20block%20of%20the%20council%20cb6e33effa5a4b75b0ba8624cbd5e179.md) ($265.73 [$25.7 / M](https://joystreamstats.live/tokenomics) tJOY)
    
    - Calculate Start Reward Validators:
    
    Use the same [guide](How%20to%20find%20the%20start%20and%20end%20block%20of%20the%20council%20cb6e33effa5a4b75b0ba8624cbd5e179.md)
    
    ![Untitled](How%20to%20fill%20a%20budget/Untitled%207.png)
    

- **Councilors**
    
    Councilors tJOY reward you can see here after the term will grade in the column [Council_tJOY_Reward](https://www.notion.so/b9ea7c6923e74539941477d1a2e083a6?pvs=21) (total) 
    

- **Daily Council Budget**
    
    To calculate the daily Council Budget you need to go to this [link](https://polkadot.js.org/apps/?rpc=wss%3A%2F%2Frpc.joystream.org%3A9944#/chainstate) to Polkadot and choose the following data:
    
    ![Untitled](How%20to%20fill%20a%20budget/Untitled%208.png)
    

- In order to calculate amounts **in dollars**, you need to check the current value of tJOY on displayed [**here**](https://joystreamstats.live/tokenomics) (ex. Start Groups Budget*Rate=x)

![Untitled](How%20to%20fill%20a%20budget/Untitled%209.png)

- **Total amount:** Funding proposal **+** Validators Rewards + Councilors Rewards + Planned Increase WG Rewards = Total amount
