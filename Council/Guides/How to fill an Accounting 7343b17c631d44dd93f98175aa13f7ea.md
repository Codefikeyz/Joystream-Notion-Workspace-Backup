# How to fill an Accounting

- Validators
    - To count the rewards for Validators, first, you need to [find the start and end block of the council term](How%20to%20find%20the%20start%20and%20end%20block%20of%20the%20council%20cb6e33effa5a4b75b0ba8624cbd5e179.md)
    
    ![Untitled](How%20to%20fill%20an%20Accounting/Untitled.png)
    
    - Put this information here:
    
    ![Untitled](How%20to%20fill%20an%20Accounting/Untitled%201.png)
    
    - Calculate an average between the start era reward and end era reward in the and of the [Guide](How%20to%20find%20the%20start%20and%20end%20block%20of%20the%20council%20cb6e33effa5a4b75b0ba8624cbd5e179.md)
    
    Example result: 
    - (45.86 + 22.50) / 2 = 34.18 k tJOY
    
    ![Untitled](How%20to%20fill%20an%20Accounting/Untitled%202.png)
    
    - Multiply the average era reward by the amount of eros in a week (168) to find these values
    
    Example result: 
    34.18  * 168 = 5 742.24 k tJOY = 5.74 M tJOY
    
    ![Untitled](How%20to%20fill%20an%20Accounting/Untitled%203.png)
    
    - To find this value click on the [link](https://joystreamstats.live/tokenomics) and indicate the **rate** at the end of the period
    
    ![Untitled](How%20to%20fill%20an%20Accounting/Untitled%204.png)
    
    - Multiply M tJOY amount by the rate:
    
    Example result: 5.74*24.9=143
    
    ![Untitled](How%20to%20fill%20an%20Accounting/Untitled%205.png)
    
- Working groups
    - Calculate the values for the working groups (tabular part):
    - To calculate the end Budget ****use [**guide](How%20to%20check%20the%20amount%20in%20the%20WG%20budgets%20863e2c26abb843cf80cba0f000783e02.md),** but use only the block end the era reward at the end of the term:
    
    ![Untitled](How%20to%20fill%20an%20Accounting/Untitled%206.png)
    
    - Write down the received information in the column **End** **Budget M tJoy**
    
    ![Untitled](How%20to%20fill%20an%20Accounting/Untitled%207.png)
    
    - To fill the Planned Increment amount (M tJOY)**,** use information from executed Budget [Proposals](https://dao.joystream.org/#/proposals/past) or Budget Allocation in the Term Plan
    - To fill the Approved amount (M tJOY), use information from executed Budget [Proposals](https://dao.joystream.org/#/proposals/past)
    
    ![Untitled](How%20to%20fill%20an%20Accounting/Untitled%208.png)
    
    - To fill the **Lead rewards use this [Guide](How%20to%20fill%20a%20budget%20ce319b767dbb4e62b4a5016b3e555205.md) (***Fill the tabular part)*
    - To calculate **Amount Spent** from the Budget use this Formula:
    
    B.start+B.approved-B.end=x, where **B** is budget 
    
    Example result: 12.21+8 - 6.10=14.11
    
- Funding proposal
    
    To calculate the Funding proposal it is necessary to add up the amounts that were executed through the Funding proposals
