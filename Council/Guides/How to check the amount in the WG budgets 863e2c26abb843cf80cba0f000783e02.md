# How to check the amount in the WG budgets

- **Calculate the starting budget for each group**

![Untitled](How%20to%20check%20the%20amount%20in%20the%20WG%20budgets/Untitled.png)

- Use the [**guide](How%20to%20find%20the%20start%20and%20end%20block%20of%20the%20council%20cb6e33effa5a4b75b0ba8624cbd5e179.md)** to find the start block of the council term, the block hash with a block number
- open this **[link](https://polkadot.js.org/apps/?rpc=wss%3A%2F%2Frpc.joystream.org%3A9944#/chainstate)**, specify the following data

![Untitled](How%20to%20check%20the%20amount%20in%20the%20WG%20budgets/Untitled%201.png)

Builders = operationsWorkingGroupAlpha

Content = contentWorkingGroup

Distribution = distributionWorkingGroup

Forum = forumWorkingGroup

HR = operationsWorkingGroupBeta

Marketing= operationsWorkingGroupGamma

Storage = storageWorkingGroup

- **Then click plus and the information will appear below (Start budget balance)**
