# Example of JSON structure for tracking governance

- link to proposal(s)
    - link to associated forum thread(s)
    - categorization
    - field: due date
    - field: working group
    - field: type
    - field: milestone(s)
- link to report(s)
    - forum thread
    - WG
    - author name
    - key metrics
    - key notes
